## Schnellstart (GitHub Pages)
1. Dieses Repository auf GitHub anlegen (z. B. `gr-it-systems-site`).
2. Dateien hochladen (oder ZIP entpacken & pushen).
3. In den Repository‑Settings **Pages** aktivieren → **Branch: main**, **/ (root)** auswählen.
4. Nach 1–2 Minuten ist die Seite live unter `https://<dein-user>.github.io/gr-it-systems-site/`.
